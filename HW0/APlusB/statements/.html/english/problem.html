1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

You are given two integers $$$a$$$ and $$$b$$$. Print $$$a+b$$$.

Input

The only line of the input contains integers $$$a$$$ and $$$b$$$ ($$$-100 \le a,b \le 100$$$).

Output

Print $$$a+b$$$.

Examples

Input
7 8
Output
15
Input
-100 100
Output
0
Input
-7 -99
Output
-106

Note

In the first example, $$$a=7$$$ and $$$b=8$$$. Thus, the answer is $$$a+b=7+8=15$$$.
